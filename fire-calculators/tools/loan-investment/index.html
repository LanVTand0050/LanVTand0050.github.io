如何解讀

比較基準

比較期間固定為貸款年限。不貸款方案會把「貸款方案需要支付的每月還款」拿去定期定額投資，因此兩邊使用相同每月現金流。

貸款方案淨資產

因為比較期間等於貸款年限，期末貸款已清償，貸款方案淨資產就是期末投資資產。

貸款投資勝率

勝率代表在所有模擬路徑中，貸款方案期末淨資產高於不貸款方案的比例。勝率高不代表風險低，仍需觀察 P10 或更差情境。

P10 / P50 / P90

P50 是中位數情境，代表一半模擬結果高於此值、一半低於此值。P10 是較保守情境，代表約 10% 結果低於此值；P90 則是較樂觀情境，代表約 90% 結果低於此值。

損益兩平報酬率

這是固定報酬假設下，讓兩種方案期末結果相同的年化報酬率。若預期報酬率低於此值，貸款投資通常不具優勢。

判讀建議：先看每月還款是否造成現金流壓力，再看 P50 差異與 P10 差異。若 P50 優勢有限，但 P10 虧損很大，代表槓桿承擔的下行情境可能不值得。

此工具僅為財務情境試算，不構成投資或借貸建議。
